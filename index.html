Open Energy Dashboard Software Contributor
06/23 - 08/23

The Open Energy Dashboard (OED) is a web application that provides a platform for organizations to manage, visualize and analyze their energy data. The application was built using React, Node.js, Express, and MySQL.

As a Software Contributor to this project, I was responsible for developing a new feature that would allow administrators to create custom checks to ensure data integrity. This feature was developed using React, Node.js, and MySQL. I also wrote an article on LinkedIn to share my experience and the impact of the feature on the project.

The feature was well-received by the community and was merged into the main branch of the project. I also received positive feedback from the project maintainer Professor Huss Lederman and my teammates for my work on this feature.
